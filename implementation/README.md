# What is Salsa20?

Salsa20 is a stream cipher designed by Daniel J. Bernstein.

Unlike block ciphers (AES), Salsa20 does not encrypt data block-by-block.

Instead, it generates a pseudorandom keystream.

Encryption:

Ciphertext = Plaintext XOR Keystream

If plaintext is all zeros,

Ciphertext = Keystream

This trick is used for extracting the Salsa20 keystream.

---

# Important Terms

## Key

Secret value used by Salsa20.

Length:

256 bits

32 bytes

Generated using:

```python
key = get_random_bytes(32)
```

---

## Nonce (Number Used Once)

Ensures different keystreams even if the same key is reused.

Length:

64 bits

8 bytes

Generated using:

```python
nonce = get_random_bytes(8)
```

---

## Counter

Represents the current Salsa20 block.

Counter = 0

↓

First 64-byte keystream block.

Counter = 1

↓

Next 64-byte block.

In this project only Counter = 0 is used.

---

## Keystream

Random-looking bytes generated by Salsa20.

One Salsa20 block produces:

64 bytes

↓

512 bits

---

# Why encrypt zero bytes?

Code:

```python
keystream = cipher.encrypt(bytes(64))
```

bytes(64)

↓

64 bytes containing only zeros.

Since

Ciphertext = Plaintext XOR Keystream

and

0 XOR X = X

the output becomes the keystream itself.

This matches the official ECRYPT test vectors.

# Important Python Functions Used

## Salsa20.new()

Creates a Salsa20 cipher object.

Example:

```python
cipher = Salsa20.new(
    key=key,
    nonce=nonce
)
```

Arguments:

key → 32 bytes

nonce → 8 bytes

---

## cipher.encrypt()

Encrypts plaintext.

Example:

```python
keystream = cipher.encrypt(bytes(64))
```

Since plaintext consists entirely of zeros,

the returned value is the Salsa20 keystream.

---

## get_random_bytes()

Generates cryptographically secure random bytes.

Example:

```python
from Crypto.Random import get_random_bytes

key = get_random_bytes(32)
```

Never use

```python
random.randint()
```

for cryptography.

---

## bytes.fromhex()

Converts hexadecimal string into bytes.

Example:

```python
key = bytes.fromhex(
    "800000000000..."
)
```

Required because Salsa20 accepts bytes, not strings.

---

## .hex()

Converts bytes into hexadecimal string.

Example:

```python
print(keystream.hex())
```

---

## format()

Used for binary conversion.

Example:

```python
format(227, "08b")
```

Output:

11100011

Meaning:

08 → Always 8 bits

b → Binary format

---

## writer.writerow()

Writes one row into CSV.

Example:

```python
writer.writerow(bits)
```

If

bits =

```python
[1,1,1,0]
```

CSV stores

1,1,1,0
